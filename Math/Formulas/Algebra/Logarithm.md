$$a^x=y == log_a y = x$$
Common bases and restricted values
a > 0 && a != 1
y > 0

The product rule
$$log_a(xy) == log_a x + log_a y$$

The quotient rule
$$log_a(\frac{x}{y}) == log_a x - log_a y$$

The power rule
$$log_a(x^n) == n log_a x$$

Euler's Number and natural logarithm
$$e = 2.718281828459045...$$
$$log_e (x) == ln(x)$$

The change of base formula, where c=10
$$log_a(b) == \frac{log_c(b)}{log_c(a)}$$
