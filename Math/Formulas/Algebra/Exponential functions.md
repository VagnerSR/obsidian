$$f(x)=ab^x$$
a = initial value/y intercept (0, a)
b = growth factor
	b>1 increasing
	0< b<1 decreasing

Domain (-∞, ∞)
Range
	(0, ∞) a>0 // +a above horizontal axis
	(-∞, 0) a<0 // -a below horizontal axis

Horizontal
	stretch/compression $$b^{kx}$$
		if 0<|k|<1 stretch
		if 1<|k| compression
	shift $$f(x)=b^{x+c}$$
	reflection $$b^x$$ $$b^{-x}$$
Vertical
	stretch/compression $$ab^{x}$$
		if 0<|a|<1 stretch
		if 1<|a| compression 
	shift $$f(x)=b^x+d$$
	reflection $$b^x$$ $$-b^{x}$$
